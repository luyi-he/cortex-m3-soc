## 1. 系统概述

### 1.1 设计目标

| 指标 | 目标值 | 单位 |
|------|--------|------|
| 工作频率 | 200 | MHz |
| 核心电压 | 1.0 | V |
| IO 电压 | 1.8 | V |
| 动态功耗 | <50 | mW @200MHz |
| 静态功耗 | <10 | mW |
| 睡眠功耗 | <100 | μW |
| 核心面积 | <2.0 | mm² (28nm) |

### 1.2 技术选型

- **CPU**: ARM Cortex-M3 r2p1 (带 FPU 可选)
- **工艺**: TSMC 28nm HPC+
- **封装**: QFN64 / LQFP64
- **EDA 工具**: DC + ICC2 + PT

## 5. 时钟架构

### 5.1 时钟源

```
                    +------------------+
                    |  External OSC    |
                    |  25 MHz ±50ppm   |
                    +--------+---------+
                             |
              +--------------+--------------+
              |                             |
    +---------v----------+        +---------v----------+
    |      Internal      |        |      External      |
    |     RC 24MHz       |        |     25MHz OSC      |
    |   (±1% accuracy)   |        |                    |
    +---------+----------+        +---------+----------+
              |                             |
              +--------------+--------------+
                             |
                    +--------v--------+
                    |   Clock Mux     |
                    |   (CLK_SRC)     |
                    +--------+--------+
                             |
                    +--------v--------+
                    |      PLL        |
                    |  (PLL_M,PLL_N)  |
                    +--------+--------+
                             | 200MHz
              +--------------+--------------+
              |              |              |
    +---------v----+ +-------v------+ +-----v------+
    |  CPU Clock   | |  AHB Clock | |  APB Clock |
    |  200MHz      | |  200MHz    | |  100MHz    |
    |  (HCLK)      | |  (HCLK)    | |  (PCLK)    |
    +--------------+ +------------+ +------------+
```

### 5.2 PLL 配置

| 参数 | 值 | 描述 |
|------|-----|------|
| PLL_M | 25 | 输入分频 |
| PLL_N | 400 | 倍频系数 |
| PLL_P | 2 | 系统时钟分频 |
| 输出频率 | 25 * 400 / 2 = 500MHz |
| HCLK | PLL/2.5 = 200MHz |
| PCLK | HCLK/2 = 100MHz |

### 5.3 时钟门控

每个外设模块独立时钟使能，通过 RCC 寄存器控制：

```verilog
// RCC_AHB1ENR (0x5000_A010)
bit[0]  GPIOAEN   - GPIO A 时钟使能
bit[1]  GPIOBEN   - GPIO B 时钟使能
bit[2]  GPIOCEN   - GPIO C 时钟使能
bit[3]  GPIODEN   - GPIO D 时钟使能
bit[4]  DMAEN     - DMA 时钟使能
bit[8]  CRCEN     - CRC 时钟使能

// RCC_APB1ENR (0x5000_A020)
bit[0]  UART0EN   - UART0 时钟使能
bit[1]  UART1EN   - UART1 时钟使能
bit[2]  TIMER0EN  - Timer0 时钟使能
bit[3]  TIMER1EN  - Timer1 时钟使能
bit[4]  TIMER2EN  - Timer2 时钟使能
bit[5]  TIMER3EN  - Timer3 时钟使能
bit[8]  WDTEN     - 看门狗时钟使能
bit[9]  RTCEN     - RTC 时钟使能
bit[10] I2C0EN    - I2C0 时钟使能
bit[11] I2C1EN    - I2C1 时钟使能
bit[12] SPI0EN    - SPI0 时钟使能
bit[13] SPI1EN    - SPI1 时钟使能
```

## 8. 总线协议

### 8.1 AHB-Lite 协议

**信号列表**:

| 信号 | 方向 | 描述 |
|------|------|------|
| HCLK | - | AHB 时钟 |
| HRESETn | - | AHB 复位 (低有效) |
| HADDR[31:0] | 主→从 | 地址总线 |
| HTRANS[1:0] | 主→从 | 传输类型 |
| HSIZE[2:0] | 主→从 | 传输大小 |
| HBURST[2:0] | 主→从 | 突发类型 |
| HPROT[3:0] | 主→从 | 保护控制 |
| HWDATA[31:0] | 主→从 | 写数据 |
| HRDATA[31:0] | 从→主 | 读数据 |
| HWRITE | 主→从 | 写指示 |
| HREADY | 从→主 | 传输完成 |
| HRESP | 从→主 | 传输响应 |

**传输类型 (HTRANS)**:

| HTRANS | 描述 |
|--------|------|
| 2'b00 | IDLE |
| 2'b01 | BUSY |
| 2'b10 | NONSEQ |
| 2'b11 | SEQ |

### 8.2 APB 协议

**信号列表**:

| 信号 | 方向 | 描述 |
|------|------|------|
| PCLK | - | APB 时钟 |
| PRESETn | - | APB 复位 |
| PADDR[31:0] | 主→从 | 地址 |
| PSEL | 主→从 | 片选 |
| PENABLE | 主→从 | 使能 |
| PWRITE | 主→从 | 写指示 |
| PWDATA[31:0] | 主→从 | 写数据 |
| PRDATA[31:0] | 从→主 | 读数据 |
| PREADY | 从→主 | 就绪 |
| PSLVERR | 从→主 | 错误 |

**APB 时序**:

```
Cycle 1: PSEL  高，PENABLE 低  (Setup)
Cycle 2: PENABLE 高            (Access)
         PREADY 高 → 完成
         PREADY 低 → 等待
```

## 10. 设计检查清单

### 10.1 RTL 检查

- [ ] 所有寄存器有复位值
- [ ] 状态机有三段式
- [ ] CDC 路径有同步器

- [ ] 时序收敛 (WNS >= 0)

- [ ] 无 DRC 违例
- [ ] 时钟树平衡

- [ ] 代码覆盖率 >95%
- [ ] 功能覆盖率 >95%
- [ ] 所有测试用例通过
