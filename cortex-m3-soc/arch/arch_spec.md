# Cortex-M3 SoC 架构规格

## 1. 系统架构

### 1.1 顶层框图

```
                    +------------------+
                    |   Cortex-M3 CPU  |
                    |  (ARM CoreSight) |
                    +--------+---------+
                             | AHB-Lite
                    +--------+---------+
                    |   AHB Matrix     |
                    +--+----+----+-----+
                       |    |    |
            +----------+    |    +----------+
            |               |               |
    +-------+-------+ +-----+------+ +------+-------+
    |   ITCM 64KB   | |  Flash     | |   AHB2APB    |
    |   SRAM        | |  Controller| |   Bridge     |
    +---------------+ +------------+ +------+-------+
                                              | APB
                                    +---------+---------+
                                    |  Peripherals      |
                                    | - GPIO            |
                                    | - UART            |
                                    | - Timer           |
                                    | - NVIC            |
                                    | - WDT             |
                                    +-------------------+
```

### 1.2 地址映射

| 起始地址 | 结束地址 | 大小 | 模块 |
|----------|----------|------|------|
| 0x0000_0000 | 0x0007_FFFF | 512KB | Flash |
| 0x2000_0000 | 0x2000_FFFF | 64KB | ITCM SRAM |
| 0x2001_0000 | 0x2001_FFFF | 64KB | DTCM SRAM |
| 0x4000_0000 | 0x4000_FFFF | 64MB | AHB 外设 |
| 0x5000_0000 | 0x5000_FFFF | 64MB | APB 外设 |
| 0xE000_0000 | 0xE00F_FFFF | 1MB | NVIC/SCS |

### 1.4 复位架构

- 上电复位 (POR)
- 外部复位引脚 (NRST)
- 看门狗复位 (WDT)
- 软件复位 (AIRCR.SYSRESETREQ)

### 1.5 功耗域

| 功耗域 | 电压 | 模块 |
|--------|------|------|
| PD_CPU | 1.0V | CPU、ITCM、DTCM |
| PD_AHB | 1.0V | AHB 总线、Flash 控制器 |
| PD_APB | 1.0V | APB 总线、外设 |
| PD_IO | 1.8V | GPIO、PAD |

## 2. 中断设计

### 2.1 NVIC 配置

- 中断源：1-240（Cortex-M3 标准）
- 优先级：4 位（16 级）
- 嵌套：支持

### 2.2 外部中断映射

| IRQ | 中断源 | 优先级 |
|-----|--------|--------|
| 0 | WDT | 0 (最高) |
| 1 | UART0 | 1 |
| 2 | Timer0 | 2 |
| 3 | GPIO | 3 |
| ... | ... | ... |

## 3. 外设规格

### 3.1 GPIO

- 端口数：4 (PA, PB, PC, PD)
- 每端口引脚：16
- 模式：输入、输出、复用、模拟
- 驱动能力：2/4/8/16mA

### 3.2 UART

- 数量：2
- 波特率：最高 5Mbps
- FIFO：16 字节
- 流控：RTS/CTS

### 3.3 Timer

- 数量：4
- 位宽：32-bit
- 模式：定时、计数、PWM、输入捕获

### 3.4 WDT

- 位宽：32-bit
- 超时范围：1ms - 65s
- 窗口模式：支持

## 4. 设计约束

### 4.1 时序约束

- 时钟频率：200MHz (5ns)
- 输入延迟：2ns
- 输出延迟：2ns
- 时序例外：False path (异步时钟域)

### 4.2 面积目标

- 核心面积：< 2mm²
- 标准单元：~50K gates

### 4.3 功耗目标

- 动态功耗：< 50mW @ 200MHz
- 静态功耗：< 10mW
- 睡眠功耗：< 100uW
