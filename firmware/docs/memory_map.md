# 内存映射说明

本文档描述 Cortex-M3 SoC 的内存布局和地址映射。

## 全局地址映射

基于架构规格文档 v1.0 Section 2.1。

```
+------------------+------------+------------+---------+--------+
| 区域             | 起始地址   | 结束地址   | 大小    | 总线   |
+------------------+------------+------------+---------+--------+
| Code (Flash)     | 0x00000000 | 0x0007FFFF | 512KB   | AHB    |
| SRAM             | 0x20000000 | 0x2001FFFF | 128KB   | AHB    |
| AHB 外设         | 0x40000000 | 0x400FFFFF | 1MB     | AHB    |
| APB 外设         | 0x50000000 | 0x500FFFFF | 1MB     | APB    |
| PPB (NVIC)       | 0xE0000000 | 0xE00FFFFF | 1MB     | AHB    |
+------------------+------------+------------+---------+--------+
```

## Flash 映射 (0x00000000)

```
+------------------+------------+------------+---------+
| 区域             | 起始地址   | 大小       | 描述    |
+------------------+------------+------------+---------+
| Vector Table     | 0x00000000 | 256B       | 中断向量表 |
| Bootloader       | 0x00000100 | 4KB        | 引导程序  |
| Application      | 0x00001000 | ~508KB     | 用户程序  |
+------------------+------------+------------+---------+
```

### 向量表布局

| 偏移 | 内容 | 描述 |
|------|------|------|
| 0x00 | MSP | 主栈指针初始值 |
| 0x04 | Reset_Handler | 复位中断向量 |
| 0x08 | NMI_Handler | NMI 中断向量 |
| 0x0C | HardFault_Handler | 硬件错误中断 |
| ... | ... | ... |
| 0x3C | WDT_Handler | 看门狗中断 |
| 0x40 | UART0_Handler | UART0 中断 |

## SRAM 映射 (0x20000000)

```
+------------------+------------+------------+---------+
| 区域             | 起始地址   | 大小       | 描述    |
+------------------+------------+------------+---------+
| .data            | 0x20000000 | ~8KB       | 初始化数据 |
| .bss             | 0x20002000 | ~8KB       | 未初始化数据 |
| Heap             | 0x20004000 | ~4KB       | 堆       |
| Stack            | 0x2001E000 | ~8KB       | 栈（向下增长） |
+------------------+------------+------------+---------+
```

### 链接脚本定义

```ld
MEMORY
{
  FLASH (rx)  : ORIGIN = 0x00000000, LENGTH = 512K
  SRAM  (rwx) : ORIGIN = 0x20000000, LENGTH = 128K
}

_Min_Stack_Size = 0x2000;  /* 8KB */
_Min_Heap_Size = 0x1000;   /* 4KB */
```

## 位带操作 (Bit-Band)

Cortex-M3 支持位带操作，可以将单个位作为原子操作。

### SRAM 位带别名区

```
位带区：0x20000000 - 0x200FFFFF (1MB)
别名区：0x22000000 - 0x23FFFFFF (32MB)

位带别名地址计算公式：
bit_word_addr = bit_band_base + (byte_offset × 32) + (bit_number × 4)

其中：
- bit_band_base = 0x22000000
- byte_offset = 目标地址 - 0x20000000
- bit_number = 0-7
```

### 示例

```c
/* 普通方式 */
#define GPIOA_ODR (*(volatile uint32_t *)0x50000014)
GPIOA_ODR |= (1 << 5);  /* 设置 PA5 */

/* 位带方式 (仅 SRAM 支持) */
#define BITBAND(addr, bit) \
    (*(volatile uint32_t *)(0x22000000 + ((addr - 0x20000000) * 32) + (bit * 4)))

BITBAND(0x20000000, 0) = 1;  /* 原子设置 0x20000000 的第 0 位 */
```

## 内存保护

Cortex-M3 支持 MPU (Memory Protection Unit)，可以配置区域访问权限。

### 典型配置

| 区域 | 地址范围 | 权限 | 描述 |
|------|----------|------|------|
| Flash | 0x00000000-0x0007FFFF | RX | 代码区，只读执行 |
| SRAM | 0x20000000-0x2001FFFF | RWX | 数据区，读写执行 |
| PPB | 0xE0000000-0xE00FFFFF | RW | 系统寄存器 |

## 栈和堆

### 栈布局

```
高地址
+------------------+
| Stack (8KB)      |  ← MSP (0x20020000)
| (向下增长)        |
+------------------+
| Heap (4KB)       |
+------------------+
| .bss             |
+------------------+
| .data            |  ← _sdata (0x20000000)
低地址
```

### 符号定义

```ld
PROVIDE(_stack_top = .);    /* 栈顶 */
PROVIDE(_heap_end = .);      /* 堆顶 */
PROVIDE(_heap_start = .);    /* 堆底 */
PROVIDE(_ebss = .);          /* .bss 结束 */
PROVIDE(_sdata = .);         /* .data 开始 */
```
